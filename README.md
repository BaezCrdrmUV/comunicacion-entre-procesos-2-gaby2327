comunicacion-entre-procesos-2-gaby2327 created by GitHub Classroom

## Actividad 4 - Descripción
Programa que permite ingresar un mensaje, conocer el PID de un proceso que lo realilzó así como tambien la fecha y hora en que escribió el mensaje.
Contiene un menú en el que solicita que se seleccione una opción, las cuales son:
* 1. Escribir un mensaje
* 2. Mostrar mensajes
* 3. Salir

## Funcionamiento
Para la ejecución del programa es necesario ejecutar el siguiente comando desde la terminal:
* dotnet run
Nota: Se recomienda ejecutarlo desde la terminal de Visual Studio Code
